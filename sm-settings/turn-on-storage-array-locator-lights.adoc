= 開啟儲存陣列定位器指示燈
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要在機櫃中找到儲存陣列的實體位置、您可以開啟其定位器（LED）指示燈。

.步驟
. 選取*功能表：設定[系統]*。
. 在「*一般*」下、按一下「*開啟儲存陣列定位器指示燈*」。
+
此時會開啟「*開啟儲存陣列定位器指示燈*」對話方塊、並開啟對應的儲存陣列定位器指示燈。

. 當您實際找到儲存陣列後、請返回對話方塊、然後選取* Turn Off*。


.結果
定位燈會關閉、對話方塊會關閉。
